# Product Context: Distraction-Free KJV Bible Reader

## Why This Project Exists
This is a complete rewrite of an existing personal KJV Bible reading web app (original: https://readthekjv.com/). The goal is to create the most focused, peaceful, and reverent Bible reading experience possible in a modern web browser.

## Core Problem Being Solved
Most Bible reading apps and websites are cluttered with:
- Distracting sidebars and navigation elements
- Scrolling interfaces that break reading flow
- Advertisements and promotional content
- Complicated features that overshadow the text itself

This app exists to honor the text by removing every possible barrier between the reader and the Word.

## Target User
- Primary: Desktop/laptop users seeking focused Bible study
- Someone who wants the feel of reading a physical, open Bible
- Users who value calm, spacious, text-centered design
- Mobile support is a bonus, not required

## User Experience Goals

### The Feel
- Like reading from a physical open Bible
- Calm, spacious, reverent presentation
- Seamless page-turning with keyboard shortcuts
- Zero distractions from the scripture text

### Key Interactions
1. **Reading**: Two-column layout with classic Bible formatting
2. **Navigation**: Keyboard-driven page turning and verse selection
3. **Search**: Smart search bar for references and full-text search
4. **Resumption**: Auto-saves position for seamless return visits

## Success Metrics
- No scrolling ever in reading area
- Perfect two-column verse flow
- Instant resume to last reading position
- Sub-second navigation between verses/pages
- Clean, legible typography with EB Garamond font

## Out of Scope
- Note-taking or highlighting features
- Social sharing functionality
- Multiple Bible translations
- Commentary or study notes
- Mobile-first design (mobile is secondary)
